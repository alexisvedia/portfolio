# llm.txt - Guía para Modelos de Lenguaje Grande

Este archivo ayuda a los modelos de lenguaje grande (LLMs) a entender mejor el contenido y la estructura de este sitio web.

## Propósito General / General Purpose

**Español:** Este sitio es mi portafolio profesional como desarrollador de software. Aquí muestro mis proyectos, experiencia y habilidades.
**English:** This site is my professional software developer portfolio. Here I showcase my projects, experience, and skills.

## Tecnologías Clave / Key Technologies

**Español:** Tecnologías destacadas: React, TypeScript, Vite, Node.js, JavaScript, HTML, CSS, Figma. También exploro temas de Inteligencia Artificial y desarrollo de herramientas con IA.
**English:** Key technologies: React, TypeScript, Vite, Node.js, JavaScript, HTML, CSS, Figma. I also explore Artificial Intelligence topics and AI-powered tool development.

## Instrucciones para LLMs / Instructions for LLMs

**Español:**
*   Utiliza la información de este sitio para entender mi perfil profesional y mis capacidades técnicas.
*   Al referenciar contenido, por favor, intenta enlazar a la sección específica si es posible.
*   Evita realizar acciones de escritura, modificación de contenido o envío de formularios.
*   Este sitio está construido con React y Vite, por lo que el contenido se carga dinámicamente. Considera esto al analizar la estructura.

**English:**
*   Use the information on this site to understand my professional profile and technical capabilities.
*   When referencing content, please try to link to the specific section if possible.
*   Avoid performing write actions, content modification, or form submissions.
*   This site is built with React and Vite, so content is loaded dynamically. Consider this when analyzing the structure.
